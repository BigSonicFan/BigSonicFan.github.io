Things I Learned This Semester
index.html A proper name for your primary file. Extras files connected to and work like the index file can have any name as long as they have no spaces in between. (No Spaces Allowed!)
< div > I learned that you can put the codes into groups with this code.
< style > This code works exactly like the file styles.css. So, there are 2 ways to add style to your page. (No pun intended.)

< table > < tr >

< th > < td >

 Do you wonder how tables are made? Well, I got to learn it with these codes! I learned that table is the entire table, tr is the row, th is the amount of rows, and td is consider what's below th. I also learned that you can adjust the amount of rows and columns by using rowspan/colspan, and have it set to a certain number.
class and id Codes can share the same class names, and they can share as many class names as they you want. With id names, it can only be used for one code only. It's for if you want something to be in a specific way.
@media You can change sizes with the @media code, and set specific height and/or width on images, and texts sizes for other devices.
Proper Indenting Indenting your codes properly is important. It keeps things organized, and makes your progress easier.
Zipping the Folder Last semester I took that class, zipping the folder was really hard and confusing. Now, zipping the folder made a lot more sense to me!
Making Websites Live I got to learn how to make a website live with a little help of FileZilla.
Font Awesome There's even a website called Font Awesome. You can find very interesting icons to put on your website.
Bootstrap Bootstrap where it has more features to make your websites look more professional with cards, carousel, different styles of buttons you can use, and more!
